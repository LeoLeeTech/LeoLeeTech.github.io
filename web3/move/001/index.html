      # 后续-关上terminal后再打开就不行了

      第二天回来, 新开terminal窗口, 准备快乐的学习. but 提示我command not found, what?
      原因: move命令没加到环境变量中, 只是临时在哪一个termainl中的seesion中生效了
      解决:

      • move使用cargo安装的, 所以先找到cargo的安装目录, 一般是在~/.cargo/bin

      • 把该路径加入到环境变量(我用的是zsh)

        • 打开配置文件vim ~/.zshrc
        • 把以下命令加入到最后export PATH=$PATH:/Users/你的用户名/.cargo/bin

      • 需要刷新配置文件, 让其生效,执行 source ~/.zshrc. 你如果不懂的话就退出terminal再打开就行了
