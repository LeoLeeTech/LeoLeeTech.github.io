本项目提供了几种方式，让你可以用 API 测试自己的前端实现：

- [在本地运行官方后端实现](#在本地运行官方后端实现)
- [使用官方演示部署的 API](#演示-api)

## 在本地运行官方后端实现

官方后端实现是开源的。  
你可以在[这里](https://github.com/realworld-apps/nitro-prisma-zod-realworld-example-app)找到 GitHub 仓库。
README 会提供在本地启动服务器的指引。

## 演示 API

本项目提供了一个公开托管的 API，用于测试你的前端实现。  
把你的 API 请求指向 `https://api.realworld.show/api` 即可！
